# Data Grouping and Aggregation Project

## Description
This project analyzes the Stack Overflow 2023 Developer Survey dataset using Pandas.

## Features
- Data cleaning and preprocessing
- Grouping by Country and Experience Level
- Salary aggregation and analysis

## Dataset
The dataset used (`survey_results_public.csv`) exceeds GitHub's 100MB file limit and is not included in this repository.

## Tools Used
- Python
- Pandas
- NumPy
